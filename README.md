

download.py
=============
Python2脚本，用来下载指定的git项目，需要下载的项目由download-list.txt列出。
1. 自动跳过下载过的项目。
1. 创建分类/项目目录。
1. 下载Github项目，放到code目录下。
1. 官方url存放到文件`url`。


download-list.txt
=================
使用'|'代表文本中的空格，格式如下：

```
分类目录|项目名称|项目GitUrl|项目官方URL
```
其中，项目官方URL可为空。
__按格式向该文件中添加，然后运行download脚本，可下载项目__


fail-list.txt
=============
脚本运行后会生成这个文件。其中保存了下载失败的项目。
下载失败的原因可能：
1. 异常
1. GitUrl不是有效的Git项目地址，可能该地址下，有多个Git项目


项目目录结构
=============
ProjectDir
	|----code
	|	 |----Github克隆下来的项目
	|----url
